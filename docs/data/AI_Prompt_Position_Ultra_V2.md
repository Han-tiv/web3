## 🔹 核心逻辑

### 1. 真空区走完止盈(权重 25%)🚀⭐⭐⭐⭐⭐

**核心理论**(猎徒-V神):
> "破位没有主力关键位了 止盈一半设置保本"
> "当你出现下面没有主力关键位了,就止盈一半 剩下的保本设置"

**真空区状态判断**:
```
真空区进行中: 价格在真空区内快速滑动 → HOLD(阻力小)
真空区即将走完: 距离下一关键位<3% → PARTIAL 50%(准备止盈)
真空区走完: 到达下一关键位 → PARTIAL 50%+保本止损
```

**实战案例**(猎庄):
```
ETH真空区: 3750 → 3629 → 3580 → 3515 → 3441 → 3329 → 3246

操作:
突破3750进入真空区 → HOLD(快速上涨)
接近3629(<3%距离) → PARTIAL 50%(准备止盈)
触及3629 → 剩余50%设保本止损在3750
```

**止盈策略**:
- 第1批: 真空区走完(到达下一关键位) → 止盈50%
- 第2批: 剩余50%保本止损,等待新突破或全部止盈

---

### 2. 主力关键位密集区止盈(权重 20%)⭐⭐⭐⭐

**核心理论**(猎徒-V神):
> "我们一看压力在94-06 eth到了主力关键位密集区 开始止盈"

**止盈策略(距离密集区)**:
- **进入密集区**: PARTIAL 30-50%(接近多个关键位聚集区,阻力大)
- **密集区边界**: PARTIAL 50-60%(压力确认有效)
- **突破密集区**: HOLD(趋势延续,移止损至密集区上沿)
- **多次触及≥3次未破**: PARTIAL 60-70%(压力太大,大概率回调)

**实战应用**:
```
识别主力关键位密集区(小区间多个关键位) →
  进入密集区 → 止盈30-50%
  突破密集区 → 移止损,继续持有
  多次触及未破 → 大部分止盈60-70%
```

---

### 3. 流动性时间窗口(权重 20%)⏰⭐⭐⭐⭐⭐

**核心发现**(猎徒-V神):
> "师傅之前说的流动性最多维持4-8h"
> "一般3根1h的阴线你就要注意止盈了"

**时间规则**:
```
持仓 <4h: 流动性充沛期 → HOLD(可继续持有)
持仓 4-8h: 流动性衰竭警戒期 → 密切关注止盈信号
持仓 >8h: 流动性结束(妖币) → FULL(强制止盈)
```

**警戒信号**:
- **连续3根1h阴线** → PARTIAL 50%(流动性衰竭信号)
- **连续5根1h阴线** → FULL(趋势反转)

**妖币特殊规则**:
> "流动性最多维持4-8h" → 妖币持仓>8h无论盈亏全部平仓

---

### 4. 反弹力度判断(权重 15%)⭐⭐⭐⭐

**量化标准**(猎庄):
```
1h实体 vs 上一根K线实体

强反弹: 1h实体 > 上一根实体50% → 有护盘,可能反转
  操作: PARTIAL 30-50%(主力护盘,减仓观望)

弱反弹: 1h实体 < 上一根实体50% → 无护盘,继续走弱
  操作: HOLD(无护盘,持仓到下一关键位)
```

**操作逻辑**:
```
到达关键位 → 观察1h内反弹 →
  IF 实体>50%上一根 → 主力护盘,止盈30-50%
  ELSE → 继续走弱,持仓到下一关键位
```

**实战应用**(猎庄):
> "关键位破位后,关注关键位的反弹力度,这1h内"
> "没有力度 破下一个关键位就会继续"

---

### 5. K线反转信号(权重 10%)📉

**1h级别(最高优先级)**:
- **1h跌幅>10%**: FULL(大跌见顶,全平)
- **1h跌>5% + 盈利>10%**: PARTIAL 70-80%(高位大跌,大部分止盈)
- **从1h最高回落>15%**: FULL(深度回调,趋势反转)

**5m级别**:
- **长上影线**(上影>实体2倍): PARTIAL 30-40%(抛压重,短期回调)
- **从5m最高回落>8%**: FULL(大幅回落,反转信号)

---

### 6. 盈利梯度参考(权重 5%)

**灵活非强制**:
- 5-8%: 考虑止盈20-30%(可选)
- 10-15%: 考虑止盈30-50%(建议)
- 15%+: **至少止盈50%**(强制)
- 20%+: **至少止盈70%**(强制)

⚠️ **重要**: 盈利梯度仅参考,真空区/密集区/流动性时间优先级更高!

---

### 7. 技术指标(权重 5%)

**辅助参考**:
- RSI >70超买 + 距离密集区<3% → 考虑止盈
- MACD死叉 + 连续3根1h阴线 → 考虑止盈

⚠️ **说明**: 技术指标仅5%辅助,不作主要决策依据

```

  → 1h大跌>5%? 5m长上影? 深度回调?

优先级6(5%): 盈利梯度
  → 盈利>15%? 盈利>20%?

优先级7(5%): 技术指标
  → RSI>70? MACD死叉?
```

## ✅ 持有条件(需全部满足 6/6)

1. ✓ **真空区未走完** - 下方仍有真空区空间(无关键位)
2. ✓ **未进入密集区** - 距离主力关键位密集区>3%
3. ✓ **流动性充沛** - 持仓<4h OR 无连续3根1h阴线
4. ✓ **无强反弹** - 1h实体<50%上一根(无护盘,继续走)
5. ✓ **无反转K线** - 无长上影、无倒V、无1h大跌>5%
6. ✓ **时间合理** - 持仓<8h(妖币)

**任意1条不满足,考虑部分/全部止盈!**

## ⚠️ 风险止损(亏损时)

**轻微亏损(0~-1.5%)**:
- 距支撑>3%: HOLD(暂时安全)
- 距支撑<3%: 准备止损(警惕破位)

**中度亏损(-1.5~-3%)**:
- 跌破Level2+放量: FULL(趋势不利,立即止损)
- 未破位: HOLD(等反弹,设自动止损)

**严重亏损(-3~-5%)**:
- 跌破Level3: FULL(坚决止损,避免更大)
- 未破: 严密监控(下一支撑破位立平)

⚠️ **铁律**: 亏损>-5%由代码自动全平!

## 🔥 特殊策略

### 利润回吐保护

**监控逻辑**:
- 峰值≥15% 且 回吐≥5% → PARTIAL 50%
- 峰值≥20% 且 回吐≥8% → PARTIAL 70%

**示例**:
- 盈利18%→13%(回吐5%) → 止盈50%
- 盈利25%→17%(回吐8%) → 止盈70%

---

### 散户套牢位识别(猎庄理论)

**定义**:
> "散户套牢位置: 1h K线在关键位附近一直震荡时间很久的位置区间"

**应对**:
- 识别长时间震荡区间(数小时)
- 该区间为散户套牢密集区
- 止损位避开该区间边界±1-2%

---

### 妖币特殊处理

**核心原则**(猎徒-V神):
> "流动性最多维持4-8h"

- 持仓>4h → 警惕流动性衰竭
- 持仓>8h → FULL(无论盈亏全部平仓)
- 盈利>10% → PARTIAL 50%(立即)
- 盈利>20% → PARTIAL 80%(至少)
- 任何反转信号 → FULL(立即)

---

### 博反弹错误警示(猎徒-V神)

**错误做法**:
> "很多人喜欢在瀑布的中间就去博反弹"
> "其实就相当于你想瀑布在半空中掉头一样"

**正确做法**:
- 不在快速下跌中盲目抄底
- 等关键位破位 + 反弹力度确认(1h实体>50%)
- "没有真空区和高点出现尽量少做"
