## 🔹 核心逻辑

### 1. 资金流向(权重 30%)💰⭐⭐⭐⭐⭐

**流入信号**:
- 24h资金净流入>0(最高优先级)
- 流动性充沛(盘面异动币种增多)
- 大单买入>55%
- 买盘主动成交>卖盘被动

**评分**:
- 满足3-4项: +3分(强流入)
- 满足2项: +2分(中等)
- 满足1项: +1分(弱流入)
- 满足0项: 0分(中性/流出)

**核心理论**(猎徒-V神):
> "流动性充沛 那么更多人能赚到"
> "流动性好了 异动会越来越多"

**市场情绪判断**:
- 情绪 vs 技术冲突: "市场情绪偏多 技术指标空" → 观望
- 存量资金判断: 有存量+突破 = 暴涨,无存量+跌破 = 暴跌

---

### 2. 主力关键位(权重 20%)⭐⭐⭐⭐

**识别标准**:
1. 1h/4h K线上下影线聚集区域
2. 前期高低点(7-30天拐点)
3. 整数关口($100,000, $3,000等)
4. 成交量放大区域

**主力关键位密集区**(核心概念):
> "主力关键位密集区 = 小价格区间内数小时震荡形成多个关键位" —— 猎庄
> "没形成主力关键位密集区不会走出趋势" —— 猎徒-V神

**交易信号**:
- ✅ **突破做多**: 突破密集区 + 1h收盘确认 + 成交量≥1.5倍 + 回踩不破
- ✅ **破位做空**: 跌破密集区 + 1h收盘确认 + 成交量≥1.5倍 + 反弹无力
- ❌ **假突破**: 突破后<5分钟回落 OR 成交量<1.3倍

**当前状态模板**:
```
关键阻力: ${上方阻力}
关键支撑: ${下方支撑}
密集区: ${密集区价格范围} (判断是否形成)

位置判断:
距离<3%: "接近密集区边界,等待突破"
>3%且在密集区内: "震荡整理,观望"
突破密集区: "趋势启动,入场"
```

---

### 3. 真空区识别(权重 15%)🚀⭐⭐⭐⭐⭐

**定义**(猎徒-V神):
> "真空区在哪里 哪里压力就小"
> "简单直接 又有真空区了"

**核心逻辑**:
- 真空区 = **无主力关键位的价格区间**
- 进入真空区 = 加速行情(上涨/下跌快速)
- 真空区走完 = 即将到达下一关键位

**实战价格示例**(猎庄 2025-11-10):
```
BTC真空区: 107400 → 106092 → 104678 → 103384 → 102525
ETH真空区: 3750 → 3629 → 3580 → 3515 → 3441 → 3329 → 3246
```

**识别方法**:
1. 画出所有主力关键位
2. 关键位之间空白区间 = 真空区
3. 真空区越大 = 加速幅度越大

**操作策略**:
- 突破后进入真空区 → 快速跟进(阻力小)
- 真空区走完(接近下一关键位) → 准备止盈
- 真空区内无关键位 → 持仓不动

**标的选择**:
> "关键位少+真空区大 = 容易操作(ZEC/AIA)"
> "关键位密集 = 难操作(NEAR)" —— 猎庄

---

### 4. 入场信号(权重 15%)⭐⭐⭐⭐

**核心原则**(猎徒-V神):
> "突破就多 不突破就是空"
> "等真正突破回踩 反弹多头 跌破就是空头"

**猎庄完整操作流程**:
```
第1步: 关键位密集区破位 → 开始介入
第2步: 到达下一个关键位 → 观察反弹力度(详见第5点)
第3步: 根据反弹力度决定持仓/减仓
```

**趋势转变判断**(猎庄实战):
```
BTC趋势转变: 108000
ETH趋势转变: 3850
→ "不突破短期内都只是反弹派发"

空头防守位:
BTC: 107400
ETH: 3750
→ 突破防守位 = 空头失败
```

**错误做法警示**(猎徒-V神):
> "很多人喜欢在瀑布的中间就去博反弹"
> "其实就相当于你想瀑布在半空中掉头一样"

**正确做法**:
- ⚠️ 不在下跌趋势中盲目抄底
- ✅ 等关键位破位 + 反弹力度确认
- ✅ "谨慎入场多头 等动能衰竭或趋势确认" —— 猎庄

---

### 5. 反弹力度判断(权重 10%)⭐⭐⭐⭐

**量化标准**(猎庄):
```
反弹力度 = 1h实体K线 vs 上一根K线实体

强反弹: 1h实体 > 上一根实体50% → 有护盘,可能反转
弱反弹: 1h实体 < 上一根实体50% → 无护盘,继续走弱
```

**操作逻辑**:
```
跌破关键位 → 观察1h内反弹 →
  IF 实体>50%上一根 → 主力护盘,减仓/观望
  ELSE → 继续走弱,持仓到下一关键位
```

**实战应用**:
> "关键位破位后,关注关键位的反弹力度,这1h内" —— 猎庄
> "1h实体高于上一个k线的一半 反弹力度大,低于 属于没有护盘 继续走弱"

### 6. 流动性/时间管理(权重 5%)⏰⭐⭐⭐

**核心发现**(猎徒-V神):
> "师傅之前说的流动性最多维持4-8h"
> "一般3根1h的阴线你就要注意止盈了"

**持仓时间规则**:
```
流动性持续: 4-8小时
山寨币持仓: ≤8h强制止盈
警戒信号: 连续3根1h阴线 → 开始止盈
```

**流动性判断**:
- 盘面异动币种多 = 流动性好
- 异动减少 = 流动性衰竭,注意止盈

---

### 7. 技术指标(权重 5%)📈

**辅助参考**(次要地位):
- RSI: 40-60健康,>70超买,<30超卖(可长时间停留,不能单独依赖)
- MACD: 金叉辅助多,死叉辅助空(滞后性强,仅辅助)
- 成交量: 突破时≥1.5倍(主流币1.3倍)

⚠️ **重要**:
> "vs的主力关键位 专克技术分析派" —— 猎徒-V神

**说明**: 核心成员极少提及技术指标,仅作5%辅助参考,不作主要决策依据

## ✅ 开仓标准

### BUY 做多(满足 4/6 条件)

**核心条件(至少 3/4)**:
1. **资金流向**(30%): 流动性充沛 + 24h净流入>0 + 大单买入>55% → +3分
2. **关键位突破**(20%): 突破主力关键位密集区 + 1h站稳 + 量≥1.5倍 → +3分
3. **真空区确认**(15%): 突破后进入上方真空区(无关键位阻力) → +2分
4. **入场信号**(15%): 突破后回踩不破 + 反弹力度强(1h实体>50%上一根) → +2分

**辅助条件(任意 1 条)**:
5. **时间管理**(5%): 持仓<4h(流动性充沛期) → +1分
6. **技术指标**(5%): RSI 45-65 + MACD金叉 → +1分

**评分逻辑**:
- 满足3核心+1辅助 且 总分≥9 → **BUY HIGH**(仓位25-30%)
- 满足3核心 且 总分≥8 → **BUY MEDIUM**(仓位15-20%)
- 否则 → **SKIP**

**风险控制**:
- 止损位: 支撑位 × 0.97(下方3%)
- 风险比例: 单笔风险 ≤ 总资金5%
- 风险收益比: 必须 ≥2:1

---

### SELL 做空(满足 4/6 条件)

**核心条件(至少 3/4)**:
1. **资金流向**(30%): 流动性衰竭 + 24h净流出<0 + 大单卖出>55% → +3分
2. **关键位破位**(20%): 跌破主力关键位密集区 + 1h确认 + 量≥1.5倍 → +3分
3. **真空区确认**(15%): 破位后进入下方真空区(无关键位支撑) → +2分
4. **入场信号**(15%): 破位后反弹无力(1h实体<50%上一根) → +2分

**辅助条件(任意 1 条)**:
5. **时间管理**(5%): 持仓<4h(流动性充沛期) → +1分
6. **技术指标**(5%): RSI 35-55 + MACD死叉 → +1分

**评分逻辑**: 同做多(镜像)

**风险控制**:
- 止损位: 阻力位 × 1.03(上方3%)
- 风险收益比: 必须 ≥2:1

---

### SKIP 观望(出现任意 1 条)

1. **密集区震荡**: 在主力关键位密集区内震荡,无明确突破
2. **瀑布中间**: 快速下跌中途(不在关键位附近) → "不在瀑布中间博反弹"
3. **信号冲突**: 情绪偏多BUT技术指标空 OR 价突破BUT资金出
4. **真空区走完**: 即将到达下一关键位,等待新信号
5. **流动性衰竭**: 持仓>8h OR 连续3根1h阴线
6. **风险收益比不足**: <2:1 OR 止损>5%

## 🔥 特殊策略

### 散户套牢位识别(猎庄理论)

**定义**:
> "散户套牢位置: 1h K线在关键位附近一直震荡时间很久的位置区间"

**应对**:
- 识别长时间震荡区间(数小时)
- 止损位避开散户套牢位边界±1-2%
- 避免被主力收割

---

### 主流币做多偏好(BTC/ETH/BNB)

- **阈值降低**: 量1.5倍→1.3倍,确认1h→15m
- **趋势延续**: BTC趋势持续3-5根1h K线,至少持有3-5h
- **回调加仓**: 回踩主力关键位不破 → 加仓首仓50%

---

### 妖币策略(24h波动>20% OR 市值<1亿)

**核心原则**(猎徒-V神):
> "流动性最多维持4-8h"

- **快进快出**: 持仓≤8h强制止盈
- **严格止损**: 3-4%(主流币5%)
- **分批建仓**: 首仓10-15%试探,确认后加至20-30%,绝不超30%
- **快速止盈**: 盈利>10%止盈50%,>20%止盈80%
