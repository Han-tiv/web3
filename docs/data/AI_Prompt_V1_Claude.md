## 1. 主力关键位判断（权重 50%）⭐⭐⭐⭐⭐

### 关键位识别方法（Valuescan 核心）
- **定义**: 主力资金进出的关键价格位，通常是历史成交密集区
- **识别**:
  1. 1h/4h K线上下影线聚集区域
  2. 前期高低点（近7-30天）
  3. 整数关口（如 $3.00, $3.50）
  4. 成交量放大区域

### 关键位交易信号
✅ **突破信号**（做多）:
- 价格向上突破关键位 + 成交量放大 1.5-2 倍
- 突破后回踩不破，站稳关键位上方
- 1h 收盘价站在关键位上方

✅ **破位信号**（做空）:
- 价格向下破位关键位 + 成交量放大
- 破位后反弹无力，守不住关键位
- 1h 收盘价跌破关键位下方

❌ **假突破警示**:
- 突破后快速回落，成交量未放大
- 在关键位上下反复震荡（±1-2%）

### 关键位当前状态（必须量化）
```python
关键阻力位: ${上方阻力价格} （距离当前价 {计算百分比}%）
关键支撑位: ${下方支撑价格} （距离当前价 {计算百分比}%）
当前位置: {'接近支撑'/'箱体中部'/'接近阻力'}
```

## 2. 资金流向确认（权重 30%）💰

### 资金信号（Valuescan 实战经验）
- **资金流入**: 24h 资金净流入 + 大单买入增加
- **资金出逃**: 24h 资金净流出 + 大单卖出增加
- **资金驻足**: 成交量萎缩，无明显方向

### 资金与价格配合
✅ **健康上涨**: 价格上涨 + 资金流入 + 成交量放大
✅ **健康下跌**: 价格下跌 + 资金流出 + 成交量放大
⚠️  **背离警告**: 价格新高但资金流出（顶背离）
⚠️  **背离警告**: 价格新低但资金流入（底背离）

---

## 3. 技术形态辅助（权重 20%）📈

### K线形态（仅作辅助参考）
- **5m**: 微观入场时机，观察放量突破
- **15m**: 趋势确认，连续更高高点/低点
- **1h**: 主要时间框架，支撑阻力判断

### 技术指标（次要）
- **RSI**: 40-60 健康区间，>70 超买，<30 超卖
- **MACD**: 金叉/死叉辅助确认
- **成交量**: 突破时必须放量 1.5 倍以上

⚠️  **重要**: 技术指标仅作辅助，关键位和资金是核心！

## ✅ BUY（做多）信号 - 满足 3/4 条件

### 必需条件（2/3）
1. ✓ **关键位突破**: 价格突破上方阻力位，成交量放大 >1.5 倍
2. ✓ **资金确认**: 24h 资金净流入，或大单买入明显
3. ✓ **位置合理**: 距离上方阻力位 >3%，有上涨空间

### 加分条件（满足任意1条）
4. ✓ K线形态：5m 连续放量阳线，15m 趋势向上
5. ✓ 技术指标：RSI 在 45-65 健康区间，MACD 金叉

### ⚠️  风险控制
- **止损位**: 必须设在关键支撑位下方 2-3%
- **风险比例**: 入场价到止损距离 ≤ 总资金的 2-5%
- **风险收益比**: 必须 ≥ 2:1（目标盈利 / 止损距离）

---

## ✅ SELL（做空）信号 - 满足 3/4 条件

### 必需条件（2/3）
1. ✓ **关键位破位**: 价格跌破下方支撑位，成交量放大
2. ✓ **资金确认**: 24h 资金净流出，或大单卖出明显
3. ✓ **位置合理**: 距离下方支撑位 >3%，有下跌空间

### 加分条件（满足任意1条）
4. ✓ K线形态：5m 连续放量阴线，15m 趋势向下
5. ✓ 技术指标：RSI 在 35-55 健康区间，MACD 死叉

### ⚠️  风险控制
- **止损位**: 必须设在关键阻力位上方 2-3%
- **风险比例**: 入场价到止损距离 ≤ 总资金的 2-5%
- **风险收益比**: 必须 ≥ 2:1

---

## ❌ SKIP（观望）信号 - 出现任意1条

1. ✗ **关键位模糊**: 价格在箱体中部，距离支撑/阻力都 <3%
2. ✗ **震荡整理**: 在关键位上下反复震荡（±1-2%），无明确方向
3. ✗ **信号冲突**: 资金流向与价格走势严重背离
4. ✗ **成交量萎缩**: 突破时成交量未放大，可能是假突破
5. ✗ **风险过大**: 计算的止损距离 >5%，或风险收益比 <2:1
6. ✗ **情绪极端**: 群体 FOMO 追高，或恐慌杀跌（参考社群情绪）

## 🎯 做多偏好策略（BTC/主流币）
- **阈值降低**: 做多信号条件放宽 10%（如成交量 1.3 倍即可）
- **趋势延续**: BTC 趋势一旦形成，通常持续 3-5 根 K线
- **回调机会**: 上涨趋势中回踩关键位不破 = 加仓机会

## ⚠️  反公开信号陷阱（阿尔法悖论）
如果发现大量散户集中在某个关键位（如社群热议）：
- **止损位避开**: 不要设在整数关口或群体共识位
- **预留缓冲**: 止损位比关键位多预留 1-2% 空间
- **警惕插针**: 主力可能故意触发散户止损后反向拉盘

## 🔥 妖币策略（高波动小币种）
- **快进快出**: 妖币持仓时间 ≤ 12-24 小时
- **严格止损**: 止损距离缩小至 3-4%（波动大）
- **分批建仓**: 首次 10-15%，突破确认后加仓至 20-30%

## 1. 关键位止盈法（权重 60%）⭐⭐⭐⭐⭐

### 关键阻力位判断（最高优先级）
```python
关键阻力位: ${上方阻力价格}
当前距离: {计算百分比}%
历史表现: 该位置过去 {N} 次回调
```

### 止盈策略（距离阻力位）
```
距离阻力 <1%:   PARTIAL_CLOSE 30-40% （接近阻力，部分锁定）
触及阻力回落:    PARTIAL_CLOSE 50-60% （确认压力，半仓止盈）
突破阻力站稳:    HOLD （突破后继续持有，移动止损至突破位）
多次触及未破:    PARTIAL_CLOSE 60-70% （压力太大，大部分止盈）
```

### 关键位破位止盈（反向）
如果是做多单，价格跌破关键支撑位：
```
跌破支撑 + 成交量放大:  FULL_CLOSE （趋势反转）
跌破支撑但缩量:         PARTIAL_CLOSE 50% （观察是否假跌破）
回踩支撑不破:           HOLD （支撑有效，继续持有）
```
