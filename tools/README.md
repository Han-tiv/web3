# Tools Directory

这个目录用于存放项目构建和开发工具。

## 目录结构

```
tools/
├── scripts/           # 项目构建脚本
├── config/           # 配置文件模板
└── archive/          # 已归档的临时脚本
```

## 使用说明

1. **scripts/** - 正式的项目构建和部署脚本
2. **config/** - 各环境的配置文件模板
3. **archive/** - 临时开发脚本的存档

## 开发规范

- 只有经过测试的正式脚本才能放在 `scripts/` 目录
- 临时调试脚本放在 `archive/` 目录
- 所有脚本都应该有注释和使用说明
