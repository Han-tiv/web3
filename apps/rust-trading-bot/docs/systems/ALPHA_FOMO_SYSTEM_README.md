## 🎯 系统设计理念

### 为什么专注Alpha/FOMO？

根据频道分析，我发现：

1. **主力资金频道特点**：
   - SOL、BNB、XRP等币种异动频繁
   - 山寨币占比远超BTC/ETH
   - 快进快出（15.3%出逃信号）
   - 适合短期投机

2. **Alpha机会**：
   - 新币上市早期获利空间大
   - 但流动性和风险需要谨慎评估

3. **FOMO信号**：
   - 快速拉升，短期爆发力强
   - 但容易追高，需严格止损

### 为什么是日内交易？

```
✅ 优势：
- 降低隔夜风险
- 快速止盈止损
- 资金利用率高
- 适合山寨币波动

⚠️ 劣势：
- 需要实时监控
- 交易频率高
- 手续费成本
```

## 💡 核心策略

### 入场策略

**Alpha币种**:
```
条件1: 资金流入信号 ✅
条件2: 主流交易所上市 ✅
条件3: 24H成交量 > 100万U ✅
条件4: 技术指标健康（RSI<70，MACD金叉）✅
→ 小仓位试探（1%）
```

**FOMO币种**:
```
条件1: 资金流入 + 涨幅>10% ✅
条件2: 突破关键位 ✅
条件3: 放量上涨（成交量>均量2倍）✅
条件4: RSI<70（未严重超买）✅
→ 中等仓位（2%）
```

### 出场策略

```
止盈策略:
- +3% → 减半仓（锁定部分利润）
- +5% → 全部清仓（主要目标）
- +10% → 超额完成（如有）

止损策略:
- -2% → 立即止损（无条件）
- 主力出逃信号 → 紧急清仓
- 4小时未突破 → 时间止损
```

### 仓位管理

```
总原则: 单笔风险≤2%，总仓位≤30%

资金分配:
├─ 活跃资金: 30% (用于交易)
│  ├─ Alpha: 10% (高风险高回报)
│  ├─ FOMO: 15% (中风险中回报)
│  └─ 预留: 5% (应急)
└─ 现金储备: 70% (等待机会)

单笔仓位:
- Alpha (新币): 1% (风险高)
- FOMO (热点): 2% (风险中)
- 主流币: 3% (风险低) - 但频道很少出现
```

## ⚠️ 重要风险提示

### 频道数据偏见

根据分析发现：
```
⚠️ 220次流入 vs 0次流出报告
→ 存在"报喜不报忧"倾向
→ 需要自己判断，不能盲从

⚠️ 15.3%主力出逃信号
→ 快进快出特征明显
→ 必须严格止损

⚠️ 山寨币为主（BTC仅12次）
→ 波动大，风险高
→ 仓位要小
```

### 交易纪律

```
❌ 千万不要：
1. 追高（涨幅>30%不进）
2. 重仓（单币种>3%）
3. 死扛（破位必止损）
4. 贪婪（达目标就走）
5. 盲从（AI建议也要验证）

✅ 必须做到：
1. 严格止损
2. 分批建仓
3. 及时止盈
4. 记录复盘
5. 控制情绪
```
