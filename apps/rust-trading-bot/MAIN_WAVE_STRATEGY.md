# 主升浪捕捉策略 (Main Wave Strategy) - 修正版

## 核心理念

**目标**: 捕捉主力资金进场后的主升浪,单笔收益 5-15% (10x杠杆 = 50-150%)

**特点**:
- 低频交易 (精准狙击,避免频繁手续费)
- 高盈亏比 (1笔主升浪 >> 10笔快进快出)
- **快速止损** (NOT 允许-3%~-5%洗盘!)
- 持有到顶 (不过早止盈)

---

## 一、主升浪的三个阶段 (修正版)

### 阶段1: 建仓震荡期 (预期 ±1% 小幅震荡)

**关键特征**:
- Valuescan首次主力资金异动信号
- **真主升浪**: 入场后30-60分钟小幅震荡 ±1%,NOT 回撤-3%到-5%!
- 成交量稳定或略微放大

**❌ 错误理解** (已废弃):
- ~~允许 -3% ~ -5% 的回调空间~~
- ~~判断'洗盘'还是'真跌'~~
- **问题**: 如果每笔都允许-3%~-5%亏损,50%失败率会让你亏惨!

**✅ 正确策略 - 快速止损**:
```
⏰ 30分钟检查点:
├─ 盈亏 < -1.5% → 立即止损 (这不是主升浪)
└─ 盈亏 ±1% 范围 → 继续观察

⏰ 60分钟检查点:
├─ 盈亏 < -2% → 立即止损 (主升浪失败)
├─ 盈亏 -1% ~ 0% → 时间止损 (启动失败)
└─ 盈亏 > +2% → 主升浪启动确认 ✅
```

**止损原则**:
- 入场30分钟 < -1.5% → 止损
- 入场60分钟 < -2% → 止损
- 入场60分钟仍在-1%~0% → 时间止损
- **核心**: 真正的主升浪不会大幅回撤!

---

### 阶段2: 拉升启动期 (+2% ~ +3%)

**特征**:
- 价格突破+2%,确认主升浪启动
- 开始放量上涨
- 5m连续2-3根阳线

**信号**: 主升浪启动确认
- ✅ 5m连续3根阳线 (实体>0.5%)
- ✅ 成交量较前期放大 30%+
- ✅ 突破短期阻力位

**操作**: 继续HOLD
- 不要在 +2% ~ +3% 就止盈
- 这只是主升浪的开始

---

### 阶段3: 主升浪期 (+3% ~ +15%+)

**特征**:
- 快速拉升,少有回调
- RSI持续 >60 甚至 >70
- 成交量持续放大
- 5m/15m持续阳线

**操作**: 继续HOLD
- RSI>70 不是卖点,是强势确认
- 小回调(回踩支撑)是加仓机会

**第1次止盈条件** (60%仓位):
- 盈利 >+8% + 出现第一个'警告信号':
  - 长上影线 >2%
  - 或 5m大阴线实体 >1%
  - 或 成交量骤增后快速萎缩

**第2次止盈** (剩余40%):
- 等待明确'顶部信号'

---

### 阶段4: 见顶回落期

**顶部信号** (立即全平):
1. **长上影线倒锤头**:
   - 上影线 >2%
   - 实体 <0.5%

2. **大阴线吞没**:
   - 5m大阴线实体 >1.5%
   - 吞没前2-3根阳线

3. **成交量衰竭**:
   - 成交量骤增后快速萎缩 >50%
   - 价格冲高回落

4. **破位下跌**:
   - 跌破近期支撑位 + 5m连续2根阴线

## 三、盈亏比数学

### ✅ 新策略 (快速止损)
```
止损: -1.5% ~ -2%
止盈: +8% ~ +15%
盈亏比: 8:2 = 4:1 到 15:2 = 7.5:1

胜率要求: 只需 25% 就能盈利!
- 4笔中1笔赢(+8%) + 3笔输(-2%) = +8% -6% = +2% ✅
- 如果胜率40%: 5笔中2笔赢(+16%) + 3笔输(-6%) = +10% ✅
```

### ❌ 旧策略 (允许-3%~-5%洗盘)
```
止损: -3% ~ -5%
止盈: +8% ~ +15%
盈亏比: 8:5 = 1.6:1

胜率要求: 需要 38% 才能盈利
- 5笔中2笔赢(+16%) + 3笔输(-12%) = +4% (勉强盈利)
- 如果胜率25%: 4笔中1笔赢(+8%) + 3笔输(-12%) = -4% ❌
```

**结论**: 新策略胜率要求更低,更容易盈利!

## 六、风险控制 (修正版)

### 止损原则 (核心)

**快速止损** (优先级最高):
- 入场30分钟 < -1.5% → 立即止损
- 入场60分钟 < -2% → 立即止损
- 成交量放大 + 破位 → 立即止损

**时间止损**:
- 持仓60分钟 + 仍在-1%~0% → 主升浪未启动,止损
- 持仓4小时 + 未盈利>1% → 兜底止损

**破位止损**:
- 任何时候 < -2% + 成交量放大 → 破位止损

### 主升浪 vs 假突破的判断

**真主升浪特征**:
- 入场后30-60分钟小幅震荡 ±1%
- 60分钟突破 +2% 以上
- 5m连续阳线 + 成交量放大30%+
- 趋势延续,很少大幅回调

**假突破特征**:
- 入场后持续下跌或长时间不涨
- 30分钟亏损 < -1.5%
- 60分钟仍在-1%~0%
- 成交量萎缩或放大后价格不涨
