# 🚀 Mermaid插件安装指南（超简单）

## ⚡ 3步安装（1分钟搞定）

### 第1步：打开扩展面板
在Windsurf中按：
```
Ctrl + Shift + X
```

### 第2步：搜索插件
在搜索框输入：
```
Markdown Preview Mermaid Support
```

### 第3步：点击安装
找到这个插件：
- 名称: **Markdown Preview Mermaid Support**
- 作者: **Matt Bierner**
- 点击绿色的 **"Install"** 按钮

等待10秒，安装完成！

---

## 🎯 立即使用

安装完成后：

1. **打开文件**
   ```
   PROGRAM_FLOW_DIAGRAMS.md
   ```

2. **按快捷键**
   ```
   Ctrl + Shift + V
   ```

3. **看到效果**
   - ✅ 系统架构图
   - ✅ 启动流程图
   - ✅ 并发任务图
   - ✅ 信号处理图
   - ✅ AI决策流程图
   - ✅ 持仓监控图
   - ✅ 等等共10个图！

---

## 🌟 分屏预览（推荐）

```
Ctrl + K, 然后按 V
```

效果：
- 左侧：Markdown代码
- 右侧：实时预览图表
- 超级方便！
