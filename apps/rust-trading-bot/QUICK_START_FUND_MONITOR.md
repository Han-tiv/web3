## 🚀 快速启动步骤

### 步骤1: 安装依赖

在 `Cargo.toml` 中添加：

```toml
[dependencies]
# 已有的依赖...
regex = "1.10"
chrono = { version = "0.4", features = ["serde"] }
```

### 步骤2: 配置环境变量

`.env` 文件:

```bash
# Telegram (必需)
TELEGRAM_API_ID=your_api_id
TELEGRAM_API_HASH=your_hash
TELEGRAM_PHONE=+86your_phone

# DeepSeek (必需)
DEEPSEEK_API_KEY=your_key
```

### 步骤3: 编译运行

```bash
# 编译
cargo build --bin fund_monitor --release

# 运行
cargo run --bin fund_monitor
```
